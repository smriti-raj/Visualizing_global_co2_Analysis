# 🌍 Global CO₂ Emissions Analysis Using SQL

## 📌 Project Overview

This project analyzes the **Global CO₂ Emissions Dataset** using **MySQL**. It demonstrates the complete SQL data analysis workflow, including data loading, cleaning, transformation, and exploratory data analysis (EDA). The project provides insights into global greenhouse gas emissions, energy consumption, population, GDP, and climate-related indicators.

---

## 📂 Dataset

**Dataset:** Global CO₂ Database

The dataset contains historical environmental and economic data for countries worldwide, including:
